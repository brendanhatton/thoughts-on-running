# Who is this guide for?
This guide is for most casual / amateur runners. If you are just starting out and can't yet run more than a few hundred metres, I suggest following the [Couch to 5km plan](https://c25k.com/).

This guide is intentionally quite basic, so will be of limited use to the other extreme end of the running spectrum - ie, elite runners, or those targeting specialised ultra running events. 

I'd like to think there is something to offer for everyone else that wants to continue progressing their running journey!
